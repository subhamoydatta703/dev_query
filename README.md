# Dev Query

Full stack application with Node.js backend and React/Vite frontend.

## Tech Stack

- **Backend**: Node.js, Express, MongoDB (Mongoose), JWT (Authentication), bcryptjs, cookie-parser.
- **Frontend**: React (Vite), Axios, React Router Dom.

## How It Works

- The backend serves a REST API handling user authentication and data management using MongoDB.
- The frontend is a single-page application (SPA) built with React that interacts with the backend API via Axios.

## Setup Instructions

### Backend

```bash
cd backend
npm install
npm start
```

### Client

```bash
cd client
npm install
npm run dev
```
